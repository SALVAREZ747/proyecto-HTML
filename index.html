LA OBSERVACIÓN

La observación es la adquisición activa de información sobre un fenómeno o fuente primaria. Los seres vivos detectan y asimilan los rasgos de un elemento utilizando sus sentidos como instrumentos principales. En los humanos esto no solo incluye la vista y todos los demás sentidos, sino también el uso de herramientas, técnicas e instrumentos de medición. El término también se puede referir a cualquier dato recogido durante esta actividad. El adquirir información de los fenómenos que rodean al observador, sea con trabajo de laboratorio o con trabajo de campo, es usualmente el primer paso del método empírico en la investigación científica. Estas observaciones llevan a la curiosisdad y al planteamiento de preguntas sobre por qué un fenómeno ocurre, o su relación con otros fenómenos. Las observaciones se pueden clasificar en cuanto a su ocurrencia, frecuencia, duración, tiempo, dimensiones cualitativas, entre otros.

HTML

HTML, significa HyperText Markup Language, es el código estándar para crear páginas web y aplicaciones en línea. Define la estructura de la información en una página web, permitiendo la inclusión de texto, imágenes, videos y otros elementos multimedia, utilizando etiquetas y elementos para organizar el contenido.

Más sobre HTML

CSS

CSS, que significa Hojas de estilo en cascada, es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. En escencia, CSS se encarga de definir cómo se deben mostrar los elementos de un página web en la pantalla, en papel, o en otros medios.

Más sobre CSS

JavaScript

JavaScript es un lenguaje de programación ligero y dinámico que se utiliza principalmente para crear contenido web interactivo. Es un componente esencial para la programación web moderna, permitiendo la creación de páginas web interactivas, efectos visuales, animaciones y mucho más.
